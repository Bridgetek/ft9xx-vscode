BridgeTek extension

Welcome to the FT9xx Extension from BridgeTek Company. This extension provides tools and utilities for working with Bridgetek FT9xx microcontrollers in Visual Studio Code. And below is the list of feature which is support on this extention.

Project creation:

  • Create Template Projects
    Easily create template projects for FT9xx microcontrollers.
  • Import Example Applications
    Import the example application form SDKs.

Project inspect:

  • Analyze Binary Size
    Analyze the binary size and display charts after build successfully.
  • Parse ELF Files
    Parse ELF files and display symbol information.

Project configuration:

  • Add Third-Party Libraries
    Add ported third-party libraries (FatFs, lwIP, ...) to your projects.
  • Add Layered Drivers
    Add layered drivers (USB Drivers) to your projects.
  • Add Hardware Libraries
    Add hardware libraries (UART, I2C, SPI, ...) to your projects.

Building, Programming and debugging:

  • Build
    Build the project for FT9xx microcontroller.
  • Flash Programming
    Flash programming for FT9xx microcontrollers via Programming Board (One-wire) or UART. For DFU programming, please use the FT9xx Programming Utilities tool.
  • Debugging
    Start Debugging section for FT9xx microcontroller.
